PROJECT ESTIMATION, TENDERING & PRICING (Deliverables)

STAGE-01:STUDY PROJECT SCOPE & DOCUMENTS:: To manage Project Procurement, it is imperative that all the project documents are thoruhghly studied and analyzed to understand the Scope, Technical Specifications, Design/Performance Requirements, etc. A compreensive list of all relevant project documents is prepared and kept handy for distribution as required. These documents include, but not limited to, Project Scope, Performance data, design data, drawings, specifications, bid queries, etc.

The following documents will be provided in soft copies (Excel/PDF Files)::
➢  Worksheet containing complete list of items with quantities as per Work Breakdown Structure or bid package, all listed in the order of CSI format.
➢  A Folder containing files of all Request for quotation packages in the form of BID PACKAGES for each subcontract/major item.
➢  A Folder containing files of all Request for quotations for all bulk items with approximate quantities. These bid packages are ready for sending to qualified vendors/subcontractors for proposals.
➢  A folder containing all quotations recieved from subcontractors & vendors.
➢  Price Analysis Worksheets.
➢  A Worksheet containing Detailed Priced Bill of Quantities for the entire project, either one complete bill or in phases/packages as required.
➢➢ This BOQ is ready for submittal after incorporating profit and other costs.
